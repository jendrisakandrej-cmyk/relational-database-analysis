# Relational Database Design & Analysis

A relational database built from scratch using a Pixar movies and user reviews dataset.

## Built With
- MySQL
- Python (mysql.connector)

## Database Structure
- `movies` table — movie ID, title, release year
- `reviews` table — user reviews linked to movies via foreign key

## Queries & Analysis
- Average user score per movie
- Total and average review counts
- Filtered queries by movie title
- JOIN queries across multiple tables
- Aggregations and subqueries
- Python script connecting to MySQL via mysql.connector for automated analysis

## What I Learned
- Designing a relational database schema from scratch
- Writing JOIN queries, aggregations, and subqueries
- Connecting a database to Python for automated querying
